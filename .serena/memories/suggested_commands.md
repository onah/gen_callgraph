# Suggested commands (Darwin/macOS)
- Build: `cargo build`
- Run checks: `./check.sh` (runs fmt, clippy, test)
- Format only: `cargo fmt`
- Lint only: `cargo clippy`
- Test: `cargo test`
- Run app (typical): `cargo run -- <args>`
- Useful shell commands: `git status`, `git diff`, `ls`, `cd`, `grep`, `find`.
